TechCare: Patient Management Dashboard
A responsive, high-performance healthcare dashboard built with React.js and Tailwind CSS. This project features real-time data visualization and a modular component architecture.

✨ Key Features
Interactive Patient Profiles: Seamlessly switch between patients to view unique health metrics, diagnostic history, and lab results.

Vitals Visualization: Dynamic Blood Pressure and Heart Rate charts powered by Chart.js.

Optimized Asset Delivery: Leveraged a localized JSON data structure and the React public directory to ensure 100% asset availability in serverless environments like Netlify.

🛠️ Tech Stack
Frontend: React (Vite), Tailwind CSS.

Charts: Chart.js / React-Chartjs-2.

Icons: Lucide-react and custom medical icons.
